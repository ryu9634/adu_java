문제. (if문만 활용)
국어점수, 영어점수, 수학점수를 입력받아 총점, 평균, 학점을 구하는 프로그램을 작성해보자.
(단, 학점은 평균기준이며 90점이상 A학점, 80점이상 B학점, 70점이상 C학점, 60점이상 D학점,
나머지는 F학점)
결과

영어점수를 입력하세요. 80
수학점수를 입력하세요. 70

국어점수 : 90점
영어점수 : 80점
수학점수 : 70점
총점 : 240점
평균 : 80점
학점 : B학점

    String str = "문자";
    if(str == "문자")
        System.out.println("문자와 같아요"); // 실행되지 않음

equals() : 문자열을 비교하는 메서드. 같으면 true, 다르면 false

    String str = "문자";
    if(str.equals("문자")){
        System.out.println("문자와 같아요");  // 실행됨
    }


    if ~ else
    if(조건식){
        조건식이 true인 경우 실행할 문장;
        ....
    }else{
        조건식이 false인 경우 실행할 문장;
        ....
    }


    if ~ else if ~ else(옵션)
    if(조건식1){
        조건식1이 true인 경우 실행할 문장;
        ....
    }else if(조건식2){
        조건식2이 true인 경우 실행할 문장;
        ....
    }else if(조건식3){
        조건식3이 true인 경우 실행할 문장;
        ....
    }else{
        모든 조건식이 false인 경우 실행할 문장;
        ....
    }

switch 문
- if문과 비슷하지만 좀 더 정형화된 형태
- 특정 값과 일치하는 값을 찾아 실행

    switch(변수){
        case 값1:
            값1과 변수가 같을 경우 실행할 문장;
            ...
            break;
        case 값2:
            값2과 변수가 같을 경우 실행할 문장;
            ...
            break;
        case 값3:
            값3과 변수가 같을 경우 실행할 문장;
            ...
            break;
        default:
            값이 변수와 모두 다를 경우 실행할 문장;
            ...
    }
