1.상속이란?

특정 클래스를 구성할 때 기존의 클래스의 데이터와 메소드를 부모 클래스에서 그대로 물려받아 새로운 클래스를 생성한다.
상속의 장점으로는
1.기존에 작성된 클래스를 재활용할 수 있습니다.
2.중복으로 사용되는 코드를 부모코드에 작성시 해당 코드를 적지 않아도 됩니다.
3.클래스를 계층적인 관계로 만들어서 다양한 코드를 구상할수 있습니다.
자바에서 상속하는 법
접근 제어자  클래스명 extends 부모클래스명 {
    클래스 설계
    ...
    }

2.접근제어자란?

- 외부에 공개되는 수준을 의미
- public , protected , default, private
    public : 모든 외부에서 직접 접근하거나 호출
    protected : 현재 클래스와 동일한 패키지거나 다른 패키지더라도 상속시 접근하거나 호출
    default : 현재 클래스와 동일한 패키지 내에서만 접근하거나 호출(접근제어자를 쓰지않을경우 default)
    private : 현재 클래스의  {    } 바깥쪽에서는 절대 접근하거나 호출할 수 없음

 3.super 키워드
 - 부모 클래스로부터 상속받은 멤버인 필드나 메소드를 자식 클래스에서 사용하기 위해 필요한 참조 변수

 4.this
 -인스턴스가 바로 자기 자신을 참조하는데 사용하는 참조 변수 입니다.
 this참조변수는 해당 인스턴스의 주소를 가리키고 있습니다.
 -생성자의 매개변수 이름 인스턴스 변수의 이름이 같을 경우에는 인스턴스 변수 앞에 this 키워드를 구분해야 합니다.
